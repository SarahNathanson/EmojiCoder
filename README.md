# EmojiCoder
Compiler for an esoteric language coded in emoji. 

Lines of code are formatted as [command][parameter][new-line]

Commands:
👇: push onto the stack
👆: pop off the stack

To be added:
✋: output top item on stack
👋:	output parameter

Parameters:
As binary where 🌚=0 and 🌝=1
Ex: 🌚🌝🌚🌚🌚🌚🌚🌝
	=0100 0001
	=65
	="A"
